## 常用语言元素
- ```int()``` 整数
- ```float()``` 浮点数
- ```str()``` 字符串
- 以及bool值为True和False
> 可以通过```print(type())```得知元素类型
## 运算符类型
### 算术运算符
- 除去普通的加减乘除，以及%的余数，//的整除
### 逻辑运算符
- not ，or， and，  is，  is not
``` 
flag0 = 1==1
flag1 = 2 > 3
flag2 = flag0 and flag1
flag3 = not (flag0 and flag1)
#flag2 为False
#flag3 为Ture
print(flag1 is True) # True
print(flag2 is not False)
```
>　以及math函数，math.pi
